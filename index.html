wav2vec 2.0（Meta AI, 2020）是一套自監督語音表示學習框架，透過大量未標記音訊學習語音特徵，再以少量標記資料 fine-tune，即可達到優秀的語音辨識效果。

1
Feature Encoder（CNN）
7 層 1D 卷積網路，接收原始 16kHz PCM 波形，每 20ms 輸出一個 512-dim 局部特徵向量。不需要手工特徵（MFCC 等）。
2
Quantization（訓練階段）
Gumbel-Softmax 將連續特徵映射到離散「語音單元」（codebook），做為對比學習目標。推理時跳過此步。
3
Transformer Encoder（24 層）
全域自注意力建立上下文表示，每幀都能「看到」整段音訊，捕捉長距離音韻依賴。large 模型共 317M 參數。
4
CTC Head + Greedy Decode
線性層將每幀映射到字符機率，再以 CTC（Connectionist Temporal Classification）處理對齊問題，最終輸出文字序列。

本系統推理程式碼（server.py核心片段）：

# 1. 讀取音訊並重新取樣至 16kHz
speech, _ = librosa.load(audio_buffer, sr=16000, mono=True)

# 2. 正規化輸入（Z-score by processor）
inputs = processor(speech, sampling_rate=16000, return_tensors="pt")

# 3. 模型前向傳播
with torch.no_grad():
    logits = model(**inputs).logits   # [1, time_frames, vocab_size]

# 4. CTC greedy decode
predicted_ids = torch.argmax(logits, dim=-1)
text = processor.batch_decode(predicted_ids)[0]
